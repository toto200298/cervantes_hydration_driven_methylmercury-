# cervantes_hydration_driven_methylmercury-
Thermodynamic decomposition of methylmercury halide stability shows that halide hydration, not intrinsic Hg–X affinity, governs aqueous speciation. Gas-phase affinities span 61.9 kcal/mol but compress to 9.7 kcal/mol in solution via solvation compensation. ΔGsolv(X⁻) is the principal descriptor linking hydration to stability.
